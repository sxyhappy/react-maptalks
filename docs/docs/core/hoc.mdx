## createLayer
创建 maptalks layer HOC, eg: ImageLayer, WMSLayer
### 函数签名
```ts
declare function createLayer<
    P extends LayerOptions,
    T extends Layer
>(Component: FC<P>): ForwardRefExoticComponent<PropsWithoutRef<P> & RefAttributes<T>>
```

### 参数
1. P component Props 继承 maptalks 的 LayerOption
2. T Ref current对象, 继承 maptalks Layer
